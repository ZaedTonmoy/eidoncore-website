Getting started

New to Eidoncore? Follow these steps to set up your agency workspace.

1

Create your workspace

Register at register.eidoncore.com. Choose your agency name and subdomain. You'll get a 14-day free PRO trial.

2

Invite your team

Go to Team → Invite Member. Assign roles (Owner, Admin, Project Manager, Staff). Team members get branded invitation emails.

3

Add your first client

Navigate to CRM → New Organization. Add contacts, set health scores, and start tracking follow-ups.

4

Create a project

Go to Projects → New Project. Set budgets, milestones, and assign team members. Then add tasks via Kanban or List view.
