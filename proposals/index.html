Negotiation built in

  • Accept & sign — select optional items, sign with typed or drawn signature
  • Decline — with a reason sent back to the agency
  • Counter-propose — submit requested changes for agency review
  • Request extension — ask for more time before expiry
  • Send reminders — nudge clients who haven't responded
