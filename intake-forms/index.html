Field Types

27 field types across 6 categories

From simple text inputs to advanced repeaters — every question type you could need to onboard a client properly.

  • Text: Short text, textarea, email, phone, URL, number, address, currency
  • Selection: Dropdown, multi-select, checkbox, radio, toggle
  • Date & Time: Date picker, time, datetime
  • Rich: File upload, image upload, rating, signature, color picker, slider
  • Layout: Headings, paragraphs, dividers, hidden fields
  • Advanced: Repeater groups with configurable sub-fields
Multi-Page

Step wizards for complex onboarding

Break long questionnaires into logical pages. Clients see numbered progress indicators and per-page validation — ensuring nothing gets missed before moving to the next step.

  • Add and organize pages with tab bar
  • Numbered step wizard with progress tracking
  • Per-page required field validation
  • Submit button on the last page only
  • Ideal for Company Info → Scope → Brand → Technical sections
Smart Logic

Show fields based on answers

Not every client needs every question. Use conditional visibility to show or hide fields based on previous responses — keeping forms clean and focused.

  • Select a source field, operator, and trigger value
  • Supports equals, not equals, contains, and more
  • Example: Show "Custom Budget" only when "Budget Range" equals "Custom"
  • Reduces form fatigue and improves completion rates
Validation

Enforce input quality automatically

Every field supports granular validation rules to ensure you receive clean, usable data from clients — no more back-and-forth asking for missing information.

  • Min/max character length for text fields
  • Regex pattern matching for structured inputs
  • File type and size restrictions for uploads
  • Numeric range limits for sliders and numbers
  • Configurable star count (3-10) for ratings
  • Labels, placeholders, and help text per field
Service Integration

Assign forms to services

Link intake forms to your service catalog. When a client purchases a service, the form appears automatically on the confirmation page — zero manual follow-up needed.

  • One form can serve multiple services
  • Each service links to exactly one form
  • Works with project-based services and digital products
  • 24-hour nudge notification if form isn't submitted
  • Unlinking a form from a service doesn't delete it
