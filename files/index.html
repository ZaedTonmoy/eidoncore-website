File Types

Every format your agency needs

From design files to contracts to video — Eidoncore accepts all common file types with automatic type validation and safe upload handling.

  • Images: JPG, PNG, GIF, SVG, WebP
  • Documents: PDF, DOC, DOCX, TXT
  • Spreadsheets: XLS, XLSX, CSV
  • Archives: ZIP
  • Media: MP4, MP3, WAV, WebM
  • 10MB max per file, 5MB for branding images
File Browser

See every file across your agency

The central file browser aggregates all files from projects, tasks, and direct uploads into one searchable, filterable interface with stats at the top.

  • Stats dashboard: total files, total size, images, documents
  • Context breadcrumbs: Client → Project → Task
  • Owner info and upload dates
  • Download via signed URLs
  • Bulk delete with select-all support
Views

Three ways to browse your files

Switch between view modes depending on how you work — compact list, detail cards, or a visual thumbnail grid.

  • List: Compact rows — name, type, size, date, context
  • Detail: Thumbnail previews with uploader info
  • Grid: Large thumbnail grid for visual browsing
  • Filter sidebar: source, type, client
  • Combine filters with search by file name
Projects & Tasks

Files live where the work happens

Files uploaded to projects appear in the project's Files tab. Task attachments live on the task and link back to the parent project. Pin important files for quick access.

  • Project Files tab with all members access
  • Pinned files on project overview for quick access
  • Task drawer attachments section
  • Cross-linked: task files also appear in project
  • Upload notifications to all project members
